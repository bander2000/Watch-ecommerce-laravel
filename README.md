# Watch Ecommerce Website - Laravel

# Technologies:
-  **Language**: PHP
-  **Framework**: Laravel 8.x
-  **css Framework**: Bootstrap 4
-  **css preprocessor**: sass
-  **Database**: My Sql
-  **Text Editor** : visual studio

# Features:
- stripe payment integration
- Voyager admin panel
- Applay copoun 
- accurate calculation of tax and discount and total
- clean ui
